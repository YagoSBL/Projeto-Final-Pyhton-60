# Sistema CRM Yasbel Leads

# Funcionamento na máquina:
  Para utilizar esse programa é necessário que o Python esteja instalado em sua máquina, sistemas baseados em OS Linux já possuem essa aplicação.
  Para Instalar o python na máquina: Vá até o endereço: https://python.org.br/instalacao-windows/ e selecione o link fornecido para instalar de acordo com as propriedades da sua máquina
  A utilização desse código requer a biblioteca Custom Tkinter. A biblioteca pode já estar inclusa na sua versão python, para verificação siga os passos a seguir: Abra o prompt de comando e digite o código pip install customtkinter para garantir que já está instalado

# Prerequesitos do sistema:
  É necessário que o sistema tenha Python e CustomTkinter instalado na máquina

# Tecnologias utilizadas:
  Para esse projeto foi elaborado na linguagem python, usando biblioteca tkinter e versões customizadas, utilizando ferramentas como CRUD, Listas para melhor visualização do resultado esperado
  A versão do python usada é a 3.10
